Definition

Tree traversal is the process of visiting each node in a tree exactly once in some order. Depth First Tree Traversal is an algorthm that is used to traverse through a tree.

Depth First Traversal has three techniques; Preorder, Inorder and PostOrder. Preorder reads from root ->left ->right. Inorder reads from left ->root ->right whilst Postorder reads from left-> right->root.

Trees are better explained with the help of diagram for manual traversal.
